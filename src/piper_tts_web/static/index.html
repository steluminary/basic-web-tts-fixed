Basic TTS is a fast, free, and easy-to-use text-to-speech tool originally created by Patrick Metzger. After the original project became outdated and no longer functioned reliably due to Firebase-related issues, it was forked and minimally updated so it could continue to operate as a combined Piper host and web GUI for simple local or offline Docker-based AI text-to-speech use.

Documentation for easy Docker deployment will be added to the project page soon. The project continues to focus on CPU-based conversion of text into natural-sounding speech using high-quality English voices. Just type, select a voice, and listen or download your audio. It is useful for accessibility, content creation, testing, and other local TTS workflows.
